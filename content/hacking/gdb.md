## Registers

Information about registers

## Instructions

* Change the flavor of assembly `set disassembly-flavor intel`
* `disassembly main`
* Step instruction `si`
* Next instruction `ni`
* Set breakpoint `b *<address>`
* Set value of register `set $eax=0`
* Show register values `info registers`
